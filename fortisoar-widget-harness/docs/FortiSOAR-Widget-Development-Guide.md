# FortiSOAR Widget Development Guide (reference)

Fortinet's official **FortiSOAR Widget Development** guide (PDF) is **not
redistributed** with this repository — it is Fortinet copyrighted material.

Obtain it from your own licensed source:

- the Fortinet documentation portal (https://docs.fortinet.com), or
- your licensed FortiSOAR instance / Fortinet account team.

Then keep your local copy out of git (drop it in `docs/`; the repo ignores
`docs/*.pdf`).

`docs/KNOWLEDGEBASE.md` in this repo distills and extends the guide for the
widget-building workflows this harness supports.
